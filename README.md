widenbot-coolface
----

An (work-in-progress) coolface plugin for widenbot.

# Commands

```
!coolface
```

# Installation

1. Add dependency to your bot project:

```
npm install --save widenbot-coolface
```

2. Enable in config `plugins`:

```
module.exports = {
    plugins : {
        // ...
        coolface: {}
    }
}
```
